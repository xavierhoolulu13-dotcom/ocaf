# OCAF‑Made — DFY Agents On Demand | A mobile‑first, browser‑native agent factory that generates fully structured, ready‑to‑deploy AI agents from simple JSON templates. Load templates, render dynamic forms, generate complete agent definitions, output workflows, logic, and exportable structures. Repo structure: /templates (DFY agent JSON), /factory (index.html, factory-ui.js, agent-generator.js, self-builder.js), /deploy (deployment scripts), /branding (logo + theme), README.md. How it works: select template → fill form → factory builds agent object → output ready for integration/export/deployment. Included DFY agents: Tax Revenue Facilitator, Repo Builder & Deployer. Add templates by dropping JSON into /templates following the structure {name, description, inputs, outputs, workflows, ui}. Designed for GitHub Pages deployment. MIT License. Vision: a self‑building, agent‑driven ecosystem where repos generate repos, agents generate agents, and users deploy complete systems with a single action.
